# Najma Begum Portfolio

A professional and responsive personal portfolio website built to showcase my background, education, internships, skills, certificates, and projects.

## Features
- Modern and clean UI
- Responsive design for desktop and mobile
- Sections for About, Experience, Projects, Education, Skills, and Contact
- Project cards linked to GitHub repositories
- Easy to customize with your own details and links

## Project Structure
- index.html — Main structure of the portfolio
- styles.css — Styling and visual design
- script.js — Dynamic project cards and mobile navigation

## Preview Locally
1. Open the project folder
2. Start a local server
3. Open http://localhost:8000/ in your browser

Example:
```bash
python -m http.server 8000
```
